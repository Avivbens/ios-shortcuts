## Overview

The **Translate Text Shortcut** is an iOS shortcut that translates text into your chosen language using AI. It accepts input from the **Share Sheet**, falls back to the **clipboard**, or lets you type text manually. Pick a target language and the shortcut returns the translation — ready to copy, share, or read aloud.

---

## Features

- **Share Sheet Integration**: Trigger directly from any app that supports sharing text.
- **Clipboard Fallback**: When invoked without input, automatically reads from the clipboard.
- **Free-Style Input**: Prefer to type? Choose the manual input option and enter any text.
- **Multiple Target Languages**: Translate into Hebrew, English, Spanish, Italian, German, Portuguese, or Dutch.
- **AI-Powered**: Uses an AI model for context-aware, natural-sounding translations.
- **Flexible Invocation**: Run from the Shortcuts app, Home Screen, Siri, or the Share Sheet.

---

## Requirements

- An **iOS device** running iOS 17 or later (iOS 18+ recommended).
- The **Shortcuts** app (pre-installed on most iOS devices).
- **Internet connection** (translation requires sending content to an AI service).
- **ChatGPT app** installed and signed in, used to query the AI model.

## Usage

1. **From the Share Sheet**

    - Select text in any app (Safari, Mail, Notes, Messages, etc.).
    - Tap **Share** → **Translate Text**.

1. **From the Clipboard**

    - Copy any text.
    - Run the shortcut and choose **Clipboard** when prompted.

1. **Free-Style Input**

    - Run the shortcut and choose **Free-Style**.
    - Type the text you want to translate.

1. **Pick the Target Language**

    - Choose one of the supported languages from the list.

1. **Get the Translation**
    - The shortcut returns the translated text for you to copy, share, or use further.

## Configuration

1. **Add / Remove Languages**

    - Edit the **Target language** list inside the shortcut to add or remove options.

1. **Default Input Source**

    - The shortcut offers a menu (Clipboard / Free-Style) on each run. You can hard-code a default by removing the menu and wiring the input directly.

---

## Limitations & Privacy Considerations

- **Text Data**: The text you translate is sent to an AI service. Make sure you trust the provider and understand their data usage policies.
- **Accuracy**: AI translations may not always be perfectly accurate or idiomatic. Review before sending sensitive material.
- **Sensitive Content**: Be cautious when translating confidential or personal information.
- **Network Dependency**: A stable internet connection is required.
